== Description ==

The FixturesLive Hockey League Plugin connects your site to FixturesLive and provides a set of ad-free pages that utilises web services to populate your website. The plugin comes with 4 widgets and an intuitve interface for adding competitions to your website. 
The Plugin Also utilises it's own templates allowing webmasters to tailor the look to meet their requirements. 

Please note: you will require a FixturesLive Data Key and League ID which can be acquired from contacting FixturesLive.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "Fixtureslive Hockey League Plugin".
3. Click install.
4. Click activate.
5. Click FixturesLive.
6. Enter your FixturesLive Data Key and LeagueID.
7. Follow the install wizard
